## 📝 作业管理

### 1.1 查看作业列表
**功能描述**: 查看所有作业，支持筛选和分类

**使用方法**:
1. 登录后进入主页
2. 使用顶部标签切换：
   - **全部** - 显示所有作业
   - **待完成** - 只显示未提交的作业
   - **已完成** - 只显示已提交的作业

**作业卡片信息**:
- 📌 作业标题和描述
- 📅 截止时间
- 🏷️ 难度标签（基础/进阶/挑战）
- ✅ 完成状态
- ⏰ 剩余天数提醒

### 1.2 查看作业详情
**功能描述**: 查看作业的完整要求和参考资料

**详情内容**:
- 📄 **作业内容** - 详细的作业要求说明
- 📊 **批改结果** - 得分、分项评分、教师评语
- 📚 **参考资料** - 课件、示例代码、视频教程
- 💬 **在线答疑** - 直接向教师提问
- 📝 **我的提交** - 查看已提交的内容

### 1.3 提交作业
**功能描述**: 支持两种提交方式

#### 方式1: 代码提交
1. 点击"查看作业"按钮
2. 在代码编辑器中编写代码
3. 实时预览效果
4. 点击"提交作业"按钮

**支持的文件类型**:
- HTML文件
- CSS文件
- JavaScript文件

#### 方式2: 文档提交
1. 点击"文档提交"选项
2. 输入作业内容（支持Markdown格式）
3. 上传附件（Word、PDF、图片等）
4. 点击"提交作业"按钮

**文档提交特点**:
- 📝 富文本编辑器
- 📎 支持多文件上传
- 🖱️ 支持拖拽上传
- 📏 单文件最大50MB

### 1.4 查看提交记录
**功能描述**: 查看已提交作业的详细信息

**可查看内容**:
- ⏰ 提交时间
- 📄 代码文件列表
- 👁️ 查看代码按钮（在编辑器中打开）
- 📎 文档内容和附件
- 🔄 重做次数统计

### 1.5 申请重做作业
**功能描述**: 对不满意的作业申请重新提交

**使用流程**:
1. 在已完成作业详情页
2. 点击"申请重做"按钮
3. 填写重做原因（必填）
4. 提交申请等待教师审核

**重做规则**:
- 🔢 每个作业最多重做3次
- ✅ 需要教师批准
- 🔄 批准后分数清零，可重新提交
- 📧 审核结果会通过通知告知

## 💡 使用技巧

### 技巧1: 高效完成作业
1. 先查看作业要求和参考资料
2. 查看示例代码学习
3. 在编辑器中边写边预览
4. 遇到问题及时在线答疑
5. 完成后仔细检查再提交

### 技巧2: 提高学习效率
1. 定期查看学习报告
2. 关注知识点掌握情况
3. 及时完成即将截止的作业
4. 充分利用学习资源库
5. 积极参与在线答疑

### 技巧3: 作业重做策略
1. 仔细阅读教师评语
2. 找出问题所在
3. 重新学习相关知识点
4. 写清楚重做原因
5. 重做时注意避免之前的错误

## ❓ 常见问题

**Q: 作业提交后可以修改吗？**
A: 可以，在截止时间前可以重新提交，系统会保存最新版本。

**Q: 重做申请被拒绝怎么办？**
A: 查看教师的拒绝原因，如有疑问可通过在线答疑询问。

**Q: 代码编辑器的代码会自动保存吗？**
A: 会的，编辑器会自动保存到浏览器本地存储。

**Q: 如何查看之前提交的代码？**
A: 在"已完成"标签页点击作业，在"我的提交"部分可以查看。

**Q: 可以下载自己提交的代码吗？**
A: 可以，点击"查看代码"后在编辑器中复制或导出。
