## 💻 代码编辑器

### 1.1 编辑器概述
**技术基础**: Monaco Editor (VS Code内核)

**核心特性**:
- 🎨 **语法高亮** - 多语言支持
- 🔍 **智能提示** - 代码自动补全
- 🐛 **错误检测** - 实时语法检查
- 📏 **代码格式化** - 一键美化
- 🔄 **自动保存** - 防止数据丢失

### 1.2 支持的语言
- **HTML** - 网页结构
- **CSS** - 样式设计
- **JavaScript** - 交互逻辑
- **JSON** - 数据格式

### 1.3 编辑器功能

#### 基础功能
- ✂️ **剪切/复制/粘贴** - 标准编辑操作
- ↩️ **撤销/重做** - Ctrl+Z / Ctrl+Y
- 🔍 **查找/替换** - Ctrl+F / Ctrl+H
- 📋 **多光标编辑** - Alt+Click
- 📝 **代码折叠** - 折叠代码块

#### 高级功能
- 🎯 **跳转到定义** - F12
- 🔍 **查找引用** - Shift+F12
- 💡 **快速修复** - Ctrl+.
- 📝 **重命名符号** - F2
- 🎨 **格式化文档** - Shift+Alt+F

### 1.4 实时预览

**预览特性**:
- ⚡ **即时更新** - 代码改变立即反映
- 📱 **响应式** - 自适应不同屏幕
- 🔄 **自动刷新** - 无需手动刷新
- 🖥️ **全屏模式** - 专注预览效果
- 📏 **控制台输出** - 查看JavaScript输出

**预览模式**:
- 👁️ **分屏模式** - 编辑器和预览并排
- 🖥️ **全屏预览** - 只显示预览
- 📱 **移动预览** - 模拟手机屏幕

### 1.5 文件管理

**文件操作**:
- ➕ **新建文件** - 创建HTML/CSS/JS文件
- 📝 **重命名** - 修改文件名
- 🗑️ **删除文件** - 移除不需要的文件
- 📂 **切换文件** - 标签页切换
- 💾 **保存文件** - Ctrl+S

**文件类型识别**:
- 📄 `.html` - HTML文件
- 🎨 `.css` - CSS样式文件
- ⚡ `.js` - JavaScript文件

## 📎 文件上传系统

### 3.1 上传方式

#### 方式1: 点击上传
1. 点击上传区域
2. 选择文件
3. 确认上传

#### 方式2: 拖拽上传
1. 拖动文件到上传区域
2. 松开鼠标
3. 自动上传

### 3.2 支持的文件类型

**课件上传**:
- 📄 PDF文档
- 📊 PPT/PPTX演示文稿
- 📝 DOC/DOCX文档

**作业提交**:
- 📄 所有文档格式
- 🖼️ 图片文件
- 📦 压缩包
- 💻 代码文件

### 3.3 上传限制

**文件大小**:
- 📏 单文件最大: 50MB
- 📊 总大小限制: 200MB

**文件数量**:
- 📋 单次最多: 10个文件
- 📊 总数不限

### 3.4 上传进度

**进度显示**:
- 📊 进度条
- 📈 百分比
- ⏱️ 预计时间
- ✅ 完成状态

**错误处理**:
- ❌ 文件过大提示
- ⚠️ 格式不支持提示
- 🔄 上传失败重试
