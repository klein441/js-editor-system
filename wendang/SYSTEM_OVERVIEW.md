# 📚 编程教学平台 - 完整功能说明

## 🎯 系统概述

这是一个功能完整的**在线编程教学管理系统**，专为编程课程设计，支持教师和学生的全流程教学互动。

### 核心特点
- 🎨 **现代化UI设计** - 卡片式布局，渐变色彩，流畅动画
- 💻 **在线代码编辑器** - 实时预览HTML/CSS/JavaScript
- 📝 **多种作业提交方式** - 支持代码提交和文档提交
- 🔄 **作业重做机制** - 学生可申请重做，教师审核
- 📊 **学习数据分析** - 完整的学习报告和统计
- 💬 **在线答疑系统** - 师生实时问答互动
- 📄 **课件管理系统** - 支持多种格式课件上传下载
- 🔔 **实时通知系统** - 重要事件即时提醒

### 技术栈
- **前端**: React + Lucide Icons
- **后端**: Node.js + Express
- **数据库**: MySQL
- **编辑器**: Monaco Editor (VS Code内核)

---

## 👥 用户角色

### 1. 学生 (Student)
- 查看和提交作业
- 在线编程练习
- 查看学习报告
- 在线答疑
- 查看课件资料

### 2. 教师 (Teacher)
- 管理学生名单
- 发布和批改作业
- 管理代码库
- 上传课件
- 回复学生问题
- 处理重做申请

## 🚀 快速开始

### 启动系统
```bash

cd backend
node index.js

# 2. 启动前端（新终端）
npm start

# 3. 访问系统
浏览器打开: http://localhost:3000
```
