## 👨‍🎓 学生使用教程

### 教程1：查看和提交作业

#### 步骤1：查看作业列表
1. 登录后进入主页
2. 看到所有作业卡片
3. 使用顶部标签筛选：
   - 全部
   - 待完成
   - 已完成

#### 步骤2：查看作业详情
1. 点击作业卡片
2. 弹出详情窗口
3. 查看：
   - 作业要求
   - 截止时间
   - 参考资料

#### 步骤3：提交作业（代码方式）
1. 点击"查看作业"按钮
2. 进入代码编辑器
3. 编写HTML/CSS/JavaScript代码
4. 实时预览效果
5. 点击"提交作业"按钮
6. 确认提交

#### 步骤4：提交作业（文档方式）
1. 在作业详情中选择"文档提交"
2. 输入作业内容
3. 上传附件（可选）
4. 点击"提交作业"
5. 等待成功提示

### 教程2：查看批改结果

#### 步骤1：进入已完成作业
1. 点击顶部"已完成"标签
2. 看到所有已提交的作业

#### 步骤2：查看详情
1. 点击作业卡片
2. 在弹窗中查看：
   - 得分
   - 教师评语
   - 提交时间
   - 代码文件

#### 步骤3：查看代码
1. 在"我的提交"部分
2. 点击"查看代码"按钮
3. 代码在编辑器中打开
4. 可以复制或学习

### 教程3：申请重做作业

#### 步骤1：打开作业详情
1. 在"已完成"标签页
2. 点击要重做的作业

#### 步骤2：提交申请
1. 找到"申请重做"按钮
2. 点击按钮
3. 填写重做原因（必填）
4. 点击"提交申请"

#### 步骤3：等待审核
1. 教师会收到通知
2. 审核结果会通过通知告知
3. 批准后可以重新提交

### 教程4：使用在线答疑

#### 步骤1：打开答疑窗口
1. 点击"在线答疑"按钮
2. 或在作业详情中点击答疑

#### 步骤2：提交问题
1. 输入问题描述
2. 尽量详细说明
3. 点击"提交问题"

#### 步骤3：查看回复
1. 教师回复后会收到通知
2. 在答疑窗口查看回复
3. 可以继续追问

### 教程5：查看学习报告

#### 步骤1：打开学习报告
1. 点击"学习报告"按钮
2. 查看详细统计

#### 步骤2：分析数据
- 作业完成情况
- 分数趋势
- 知识点掌握
- 学习时间

#### 步骤3：制定计划
- 根据报告找出薄弱点
- 重点学习相关知识
- 提高学习效率

## 👨‍🏫 教师使用教程

### 教程1：管理学生名单

#### 步骤1：查看学生列表
1. 登录教师账号
2. 点击"学生名单管理"
3. 查看所有学生

#### 步骤2：添加学生
**方式1：手动添加**
1. 点击"添加学生"
2. 填写学生信息
3. 点击"添加"

**方式2：批量导入**
1. 准备CSV文件
2. 点击"导入CSV"
3. 选择文件上传

#### 步骤3：编辑学生
1. 点击"编辑"按钮
2. 修改信息
3. 保存修改

### 教程2：管理代码库

#### 步骤1：添加代码片段
1. 进入"代码库管理"
2. 点击"添加代码"
3. 填写：
   - 标题
   - 分类
   - 代码内容
4. 点击"添加"

#### 步骤2：使用代码片段
1. 发布作业时
2. 选择"关联代码案例"
3. 选择代码片段
4. 学生打开作业时自动加载

### 教程3：发布和管理作业

#### 步骤1：发布新作业
1. 进入"作业布置"
2. 点击"发布新作业"
3. 填写作业信息：
   - 标题
   - 要求
   - 截止时间
   - 目标班级
   - 关联代码（可选）
4. 点击"发布作业"

#### 步骤2：编辑作业
1. 找到要编辑的作业
2. 点击"编辑"按钮
3. 修改信息
4. 保存修改

#### 步骤3：删除作业
1. 点击"删除"按钮
2. 确认删除
3. 作业被移除

### 教程4：批改作业

#### 步骤1：查看提交列表
1. 进入"作业批阅"
2. 查看所有提交
3. 使用筛选功能：
   - 全部
   - 待批改
   - 已批改

#### 步骤2：查看学生代码
1. 点击"查看代码"
2. 代码在编辑器中打开
3. 运行预览效果
4. 评估代码质量

#### 步骤3：打分和评语
1. 点击"批改"按钮
2. 输入分数（0-100）
3. 填写评语：
   - 指出优点
   - 说明问题
   - 给出建议
4. 点击"提交批改"

### 教程5：管理课件

#### 步骤1：上传课件
1. 进入"课件管理"
2. 点击"上传课件"
3. 填写课件信息
4. 选择文件
5. 点击"确认上传"

#### 步骤2：管理课件
1. 查看课件列表
2. 可以：
   - 预览
   - 下载
   - 编辑
   - 删除

### 教程6：在线答疑

#### 步骤1：查看学生问题
1. 进入"在线答疑管理"
2. 查看问题列表
3. 使用筛选：
   - 全部
   - 待回复
   - 已回复

#### 步骤2：回复问题
1. 点击"回复学生"
2. 输入回复内容
3. 点击"提交回复"
4. 学生会收到通知

### 教程7：处理重做申请

#### 步骤1：查看申请
1. 查看重做申请列表
2. 查看学生信息和原因

#### 步骤2：审核申请
1. 查看学生之前的提交
2. 决定是否批准
3. 填写回复意见
4. 点击"批准"或"拒绝"

## ❓ 常见问题解答

### 学生常见问题

**Q1: 忘记密码怎么办？**
A: 联系教师重置密码。

**Q2: 作业提交后可以修改吗？**
A: 可以，在截止时间前重新提交即可。

**Q3: 如何查看之前的代码？**
A: 在"已完成"标签页点击作业，查看"我的提交"。

**Q4: 重做申请被拒绝怎么办？**
A: 查看拒绝原因，通过在线答疑询问教师。

**Q5: 代码编辑器的代码会丢失吗？**
A: 不会，编辑器会自动保存到浏览器。

### 教师常见问题

**Q1: 如何批量导入学生？**
A: 准备CSV文件，使用"导入CSV"功能。

**Q2: 作业发布后可以修改吗？**
A: 可以，但会通知相关学生。

**Q3: 如何查看某个学生的所有作业？**
A: 在批阅页面使用搜索功能筛选。

**Q4: 批改后可以修改分数吗？**
A: 可以，重新打开批改窗口修改。

**Q5: 如何导出学生成绩？**
A: 目前需要手动记录，后续版本会添加导出功能。
